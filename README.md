# Customer_shopping_behavior_analysis
# 📊 Data Analytics Project

## 📌 Overview

This is an end-to-end Data Analytics project that demonstrates the complete data analysis workflow—from loading raw data to creating interactive dashboards and presenting business insights.

The project includes:
- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using MySQL
- Interactive Power BI dashboard
- Business report generation
- Project presentation created using Gamma

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Open the Jupyter Notebook

```bash
jupyter notebook
```

Run the notebook step by step to perform data loading, cleaning, and analysis.

### 4. Execute SQL Queries

Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server and execute the SQL queries located in the `sql/` folder.

### 5. Open the Power BI Dashboard

Open the `.pbix` file located in the `powerbi/` folder using Power BI Desktop.
